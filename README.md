# youtube-manual-qa-project


YouTube Web – Manual QA Testing Project

Overview
This project demonstrates manual testing skills by analyzing the core functionality of YouTube Web (desktop, Chrome). It includes a structured test plan, 12 manual test cases, and documented bug reports using clear reproduction steps and expected vs. actual results.

Contents
- TestPlan.md — Scope, objectives, environment, testing types, and data.
- TestCases.md — 12 manual test cases (positive, negative, edge cases).
- BugReports.md — Documented issues with reproduction steps.
- /screenshots — Optional UI issue screenshots.

Tools Used
- Google Chrome
- Google Docs
- Markdown for documentation

Key Skills Demonstrated
- Functional testing
- UI/UX testing
- Negative testing
- Bug reporting
- Test case design
- Documentation clarity
- Attention to detail
