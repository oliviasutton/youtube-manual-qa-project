# Test Plan – YouTube Web (Desktop, Chrome)

## 1. Objective
Verify the basic functionality and UI behavior of core YouTube Web features on desktop:
- Home page
- Search
- Video playback
- Sidebar navigation
- Channel page

Focus on manual functional and UI testing.

## 2. Scope

### In Scope
- Opening YouTube and loading the home page
- Searching for videos using the search bar
- Playing a video (play, pause, seek, fullscreen, volume)
- Checking recommendations on the home page
- Navigating to a channel page from a video
- UI behavior when resizing the browser window

### Out of Scope
- Mobile app testing
- Smart TV / console YouTube apps
- Ad system, monetization, or analytics
- Recommendation algorithms

## 3. Test Environment
- Device: Windows 10/11 laptop
- Browser: Google Chrome (latest version)
- Network: Stable Wi-Fi
- Account: Logged-out state

## 4. Testing Types
- Functional testing
- UI/UX testing
- Negative testing
- Basic compatibility (window resizing)

## 5. Test Data
Search queries:
- “lofi hip hop”
- “qa testing tutorial”
- “funny cat videos”

No sensitive data used.

## 6. Entry & Exit Criteria

### Entry Criteria
- YouTube loads in Chrome
- Internet connection available

### Exit Criteria
- All test cases executed
- All defects documented
- Summary of findings created

## 7. Tools
- Chrome browser
- Google Docs / Sheets
- Markdown for documentation
